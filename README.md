# rx2fits Waterfall Viewer
A Waterfall viewer for Nathan Towne's "rx2fits", a receiver software for 16bit resolution spectrogram output for RX888 MKII SDRs. Included in this repository is the waterfall viewer itself, "fitsview_live.py" and an installation guide .pdf including links to download the modified rx2fits that allows this viewer to work. Download the .pdf file for the links to become active.
